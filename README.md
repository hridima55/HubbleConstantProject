# HubbleConstantProject

A first-year Python project that processes data from 30 1000-row text files and calculates a value for Hubble’s constant with associated uncertainty

The provided text files containing the spectral data and the CSV file with luminosity distance data with evaluation of validity of instrumental response are in the 'Data(6)' folder.

The repository includes the main 'source code' and a PNG image of the final graph that is plotted.
